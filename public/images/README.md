# Gift Site — Image Placeholders

Place your photos here. The code will automatically use them.
All images can be JPG, WEBP, or PNG.

| Filename          | Used in       | Notes                              |
|-------------------|---------------|------------------------------------|
| hero-bg.jpg       | Hero section  | Full-screen background — landscape works best (2:1 ratio) |
| timeline-1.jpg    | Timeline      | How you met |
| timeline-2.jpg    | Timeline      | First date  |
| timeline-3.jpg    | Timeline      | Milestone 3 |
| timeline-4.jpg    | Timeline      | Milestone 4 |
| timeline-5.jpg    | Timeline      | Today / most recent |
| gallery-1.jpg     | Gallery       | Full-width hero photo (wide, 21:9) |
| gallery-2.jpg     | Gallery       | Portrait (left column) |
| gallery-3.jpg     | Gallery       | Portrait (right column) |
| gallery-4.jpg     | Gallery       | Full-width (wide, 21:9) |
| gallery-5.jpg     | Gallery       | Wide (2/3 width) |
| gallery-6.jpg     | Gallery       | Tall portrait (1/3 width) |
| closing-bg.jpg    | Closing       | Full-screen background — can reuse hero or a favourite |

Tip: Compress photos to < 500KB each using squoosh.app for fast load times.
